# vagrant

Download vagrant box:
```bash
vagrant box add bento/ubuntu-16.04
vagrant box add ubuntu/focal64
```

List all vagrant boxs
```bash
vagrant box list
```

Setup Vagrantfile for an image
```bash
vagrant init ubuntu/focal64
```

Start vagrant box:
```bash
vagrant up
```

ssh into vagrant box:
```bash
vagrant ssh
```

Shutdown system:
```bash
vagrant halt
```

Destroy vagrant box:
```bash
vagrant destroy
```
